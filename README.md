# TA-Management-System

TA-Management-System is a full-stack web application for managing Teaching Assistants (TAs) in academic courses.  
The system helps instructors and administrators efficiently manage TA information, assignments, and workflows.

The system is designed to be easy to run on different machines without committing unnecessary or generated files.

---

## ✨ Features

- Manage Teaching Assistant (TA) profiles
- Assign TAs to courses and responsibilities
- Track TA workloads and schedules
- Role-based access (Admin / Instructor / TA)
- RESTful API for frontend communication
- Environment-based configuration

## How to Run

### Backend
```bash
cd backend
go run ./cmd/main.go
```
Backend server will run at:
http://localhost:8084

### Frontend
```bash
cd frontend
npm install
npm run dev
```
Frontend application will run at:
http://localhost:3000
